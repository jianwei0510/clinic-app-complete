# FlowBoard

A modern project management demo application built with Next.js 16, TypeScript, Tailwind CSS v4, and shadcn/ui.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

The database is automatically created and seeded with demo data on first run.

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS v4** + **shadcn/ui v4**
- **SQLite** via better-sqlite3
- **Recharts** for data visualization
- **Biome** for linting/formatting

## Features

- Dashboard with stats, activity feed, and project progress
- Project management with Kanban board and list views
- Task CRUD with comments and status management
- Team directory with member profiles
- Calendar view with task due dates
- Reports with charts and analytics
- Settings page with notification preferences
